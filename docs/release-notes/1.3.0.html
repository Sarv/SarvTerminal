SarvTerminal 1.3.0

Improvements & fixes

Tabs & splits

  • Drag a tab to reorder it, or drop it onto a pane edge (left / right / top / bottom) to split — with a live green drop indicator and a grab cursor.
  • Pane titles are preserved when splitting and across save/restore.
  • The attention dot no longer appears on the tab you're already viewing.

Search & hosts

  • Fixed the command palette (⌘P) — search now filters hosts live as you type, with working arrow-key navigation.
  • One consistent search across hosts, sessions, snippets, and port forwards.
  • Save & Connect — add a host and open it in one step.

SSH

  • A failed SSH connection (e.g. wrong password) now keeps the tab open and shows a clear, plain-language error instead of closing.

Sessions & workflow

  • Reopen your last session with full split layouts, per-pane working directories, SSH reconnects, titles, and tab colors.
  • Recolor saved sessions from the list.
  • Delete actions (sessions, snippets, port forwards) now ask for confirmation.

Settings & app

  • Settings opens as a proper centered window with a native title bar.
  • Vault menu shows the live sync status (provider, version, and last-synced time).
  • More comfortable terminal padding.
  • Built-in update checks that notify you when a new version is available.
