SarvTerminal 1.0.0

First public release

The fast, GPU-accelerated terminal with a built-in SSH manager — connections, splits, snippets, and encrypted sync in one app.

Terminal

  • GPU-accelerated rendering with comfortable default padding.
  • Tabs you can drag to reorder, rename, and color-tag.
  • Split panes with drag-and-drop — drop a tab onto any edge to split left / right / top / bottom.
  • Per-pane titles that follow the connection, even after splitting.

SSH & hosts

  • Save hosts with keys, passwords, jump hosts, port forwards, and startup commands.
  • Command palette (⌘P) to search hosts and quick-connect.
  • Save & Connect — add a host and open it in one step.
  • Guided connection screen with clear, human-readable errors; failed sessions stay open so you can retry.
  • Auto-reconnect on network changes and wake from sleep.

Workflow

  • Saved Sessions — capture a full split layout (local + SSH) and reopen it any time.
  • Session restore — reopen your last window of tabs and splits on launch.
  • Snippets, port forwarding, and an SFTP file browser.
  • Serial console soon

Security & sync

  • Local data encrypted at rest using a Secure-Enclave-protected key.
  • Optional end-to-end encrypted sync via GitHub or a folder.
  • Team vaults coming soon
