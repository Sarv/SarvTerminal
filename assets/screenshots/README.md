## Refreshing a shot

Press **⇧⌘4**, tap **Space**, then click the SarvTerminal window (hold **⌥** while clicking to drop
the shadow). Keep a consistent window size and theme so the gallery stays uniform.

## ⚠️ Privacy

Before committing, scrub anything identifying — real public host IPs, private repo URLs, personal
key comments, etc. (Local `127.0.0.1` is fine.) The current shots have such fields blurred/redacted.
